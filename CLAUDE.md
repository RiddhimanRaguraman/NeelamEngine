## Build
- `UberBuildMe.Bat` — full build
- `UberCleanAll.Bat` — clean
- Solution: `Neelam\Neelam.sln` (generated by `premake5.lua`)

## Layout
- `Neelam\NeelamEngine\` — C++ runtime DLL
- `Neelam\NeelamEditor\` — C# WPF editor
- `Neelam\Framework\` — shared C++ headers
- `Neelam\Vendor\` — third-party deps
